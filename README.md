# Video Conferencing

A real-time video conferencing application built with modern web technologies for multi-user video meetings.

## üöÄ Technologies

- Frontend: React, React Router

- Backend: Express.js

- Real-time Communication: Socket.IO

- WebRTC: FreeICE for STUN servers

- Build Tool: Vite

- Language: JavaScript

## üì¶ Installation and Setup

Prerequisites

- Node.js (version 16 or higher)

- npm

### Install Dependencies

```bash
npm install
```

### Running the Project

Development (Recommended)

```bash
# Start client (Vite dev server)
npm run dev

# In a separate terminal, start the server
npm run start:dev-server
```

Production Build

```bash
npm run build
```

## üåê Application Access

After starting:

- Client: http://localhost:5173 (or other port specified by Vite)
